# Comanda-APP - Order Flow

> Sistema de gerenciamento de pedidos para restaurantes. Simples, rapido e intuitivo.

## Para quem nao e tecnico

**Para o garcom:** Abre o app no celular, seleciona a mesa e registra os pedidos. A cozinha recebe na hora, sem papel.

**Para a cozinha:** Ve os pedidos em tempo real. Marca como pronto quando finalizar.

**Para o dono:** Acompanha tudo pelo painel. Relatorios de vendas por mesa e turno.

## Funcionalidades

- Gestao de mesas (ocupada, livre, em atendimento)
- Registro de pedidos com status em tempo real
- Notificacoes push
- Historico e relatorios de faturamento
- Multi-usuario (garcom, cozinha, gerente)

## Stack Tecnica

| Tecnologia | Uso |
|-----------|-----|
| React Native + Expo | App mobile iOS/Android |
| Supabase | Banco de dados + Auth + Realtime |
| Zustand | Gerenciamento de estado |
| Push Notifications | Alertas em tempo real |
